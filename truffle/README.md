## TLDR

```sh
# make sure you're on node v18
nvm use
bun add -g truffle
truffle -v

cd dApp
truffle test
truffle develop # launches the dev repl
compile # the smart contracts
migrate # the smart contracts to the blockchain created by truffle develop



```
